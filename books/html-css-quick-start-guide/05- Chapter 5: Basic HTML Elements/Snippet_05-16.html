This is a large block of text. We are using multiple lines.

As we can see, we can use paragraphs inside div elements. This serves to break up the layout.

We can have multiple paragraphs inside a div, but they are all still seen as part of the div, decorated with the "div-example" class.

On occasion, we may want tostyle a single line of text without any breaks, so we enclose it within a span element.
